# Бізнес-план платформи оголошень про ремонт "Matadora Public Ads"

## 1. Резюме проекту

**Matadora Public Ads** — це розширення існуючої платформи Matadora, яке дозволяє звичайним користувачам (власникам квартир) розміщувати оголошення про ремонтні роботи та знаходити підрядників.

### Ключові можливості:
- Розміщення оголошень про ремонт з фото, описом, бюджетом
- Прямий контакт з підрядниками через платформу
- Система рейтингів та відгуків
- Геолокаційний пошук підрядників
- Інтеграція з існуючим CRM для підрядників

## 2. Аналіз ринку

### Цільова аудиторія:
**Власники квартир:**
- Власники нерухомості, які планують ремонт
- Люди, які шукають надійних підрядників
- Користувачі з бюджетом від 50 000 до 500 000 грн

**Підрядники:**
- Малі та середні ремонтні компанії
- Індивідуальні майстри
- Спеціалізовані бригади (електрики, сантехніки, тощо)

### Конкуренти:
- OLX (категорія Ремонт/Будівництво)
- Prom.ua
- Профі.ua
- Локальні Facebook групи

### Конкурентні переваги:
- Спеціалізація виключно на ремонті
- Інтеграція з повним CRM для підрядників
- Система верифікації підрядників
- Прозорий процес відбору через відгуки

## 3. Модель доходів

### 1. Платні розміщення для підрядників:
- **Базовий план**: 500 грн/міс — до 5 відповідей на оголошення
- **Професійний**: 1500 грн/міс — необмежені відповіді + пріоритет у пошуку
- **Преміум**: 3000 грн/міс — всі можливості + виділення в пошуку

### 2. Комісія за успішні угоди:
- 5% від суми договору (для підрядників без підписки)
- 2% від суми договору (для підписників Premium)

### 3. Платні підняття оголошень:
- Підняття в топ: 50 грн/раз
- Виділення кольором: 30 грн/раз
- Закріплення на 7 днів: 100 грн

### 4. Додаткові послуги:
- Верифікація підрядника: 200 грн (одноразово)
- Послуги юриста для договорів: 500 грн/договір
- Страхування ремонту: 1.5% від суми

## 4. Технічні вимоги

### 4.1 База даних (Supabase)

**Таблиця `public_ads`:**
```sql
CREATE TABLE public_ads (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id),
  title VARCHAR(255) NOT NULL,
  description TEXT,
  property_type VARCHAR(50), -- apartment, house, office
  area_size DECIMAL(10,2),
  address TEXT,
  city VARCHAR(100),
  latitude DECIMAL(10,8),
  longitude DECIMAL(11,8),
  budget_min DECIMAL(12,2),
  budget_max DECIMAL(12,2),
  photos TEXT[], -- масив URL фото
  phone VARCHAR(20),
  preferred_contact VARCHAR(50), -- phone, email, chat
  work_type VARCHAR(50)[], -- ['full_repair', 'partial', 'electric', 'plumbing']
  start_date DATE,
  status VARCHAR(50) DEFAULT 'active', -- active, closed, pending
  views_count INTEGER DEFAULT 0,
  responses_count INTEGER DEFAULT 0,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);
```

**Таблиця `ad_responses`:**
```sql
CREATE TABLE ad_responses (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  ad_id UUID REFERENCES public_ads(id),
  contractor_id UUID REFERENCES auth.users(id),
  message TEXT,
  estimated_price DECIMAL(12,2),
  estimated_days INTEGER,
  status VARCHAR(50) DEFAULT 'pending', -- pending, accepted, rejected
  created_at TIMESTAMP DEFAULT NOW()
);
```

**Таблиця `contractor_reviews`:**
```sql
CREATE TABLE contractor_reviews (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  ad_id UUID REFERENCES public_ads(id),
  contractor_id UUID REFERENCES auth.users(id),
  rating INTEGER CHECK (rating >= 1 AND rating <= 5),
  review TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);
```

### 4.2 API ендпоінти

```
GET    /api/public-ads          — список оголошень з фільтрами
GET    /api/public-ads/:id      — деталі оголошення
POST   /api/public-ads          — створити оголошення
PUT    /api/public-ads/:id      — оновити оголошення
DELETE /api/public-ads/:id      — видалити оголошення
POST   /api/public-ads/:id/photos — завантажити фото
POST   /api/public-ads/:id/respond — відповісти на оголошення
GET    /api/public-ads/:id/responses — відповіді на оголошення
POST   /api/contractors/:id/review — залишити відгук
```

### 4.3 Функціонал завантаження фото

Використовувати Supabase Storage:
- Бакет `public-ads-photos`
- Обмеження: до 10 фото на оголошення
- Максимальний розмір: 5MB на фото
- Формати: JPG, PNG, WEBP
- Автоматичне створення thumbnail

## 5. Маркетингова стратегія

### 5.1 Запуск (Month 1-3)
- **SEO**: оптимізація під ключові слова "ремонт квартири", "пошук підрядника"
- **Content marketing**: статті про те, як вибрати підрядника
- **Social media**: таргетована реклама в Facebook/Instagram
- **Partnerships**: співпраця з інтернет-магазинами будматеріалів

### 5.2 Зростання (Month 4-12)
- **Referral program**: бонус за запрошення підрядників
- **Local SEO**: Google My Business для кожного міста
- **Email marketing**: розсилка для власників квартир
- **Influencer marketing**: співпраця з блогерами про ремонт

### 5.3 Масштабування (Year 2+)
- **TV реклама**: локальні телеканали
- **Offline events**: виставки будівельних матеріалів
- **B2B partnerships**: співпраця з девелоперами
